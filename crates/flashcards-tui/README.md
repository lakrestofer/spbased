# flashcards-tui

The flashcard tui will communicate with the central spbased daemon to retrieve a list of items that it will review, browse, tag.

## Requirements
- [ ] see and sort all flashcards - browser view
- [ ] create flashcard - add/edit view
- [ ] edit flashcard - add/edit view
- [ ] review flashcard - review view
- [ ] context menu to always show the current bindings
